一 Linux Shell 的 "Hello World" 实战入门 "#!" 代表 Shell 脚本的起始符号 通过"#!"来指定具体文本类型的特殊标记 在其之后指定 Linux 系统中解释执行当前 Shell 文件的具体的解释器 一般是 #!/bin/bash 也可以简写为 #!/bin/sh 当执行 Shell 文件的时候需要有执行权限 使用 chmod 来改变执行权限 例如 chmod 744 who.sh 需要说明的是 一般情况下会采用形如 chmod u+x who.sh 的方式来增加执行 在 Shell 文件中可以通过 "#" 后面直接加上注释的内容来增加 shell 的可读性和维护性 Shell 脚本中其实是基本命令的组合 在交互式终端中可以写入什么内容 那么在 shell 脚本中就可以写入什么内容 同一行如果有多条指令的话 我们使用";"来分割开不同的指令 例如 ls -l;who 二 VIM 编辑器的使用 VIM 是 VI 的加强版 提供了例如执行输入 输出 查找 删除 替换 块操作等众多的文本操作的内容更为强大的是用户可以根据自己的需要对 VIM 进行定制 VIM 的几种工作模式 a) normal mode 例如 ":wq" 可以保存且退出文件 ":q"是直接退出文件 ":q!" 是退出但不保存文件 b) insert mode 一般会通过输入 "i" 来进入插入模式 修改内容后按下 esc 来切换到 normal mode
